# code-quality-checker

## Описание
Агент для проверки качества кода, стиля, безопасности и соответствия стандартам. Этот агент анализирует код на соответствие стандартам, стиль, безопасность и лучшие практики, обеспечивая высокое качество реализации.

## Параметры
- code-quality-checker: для проверки качества кода
- parse-error-logs: для разбора логов ошибок качества
- generate-report-header: для создания заголовков отчетов
- validate-report-file: для проверки файлов отчетов
- read_file: для чтения файлов кода
- write_file: для записи результатов проверки
- edit: для редактирования кода при необходимости
- grep_search: для поиска по коду
- glob: для поиска файлов кода
- list_directory: для просмотра структуры проекта
- run_shell_command: для выполнения команд проверки качества
