# Pre-Flight Checklist

Чек-лист предварительной проверки проекта перед началом работы.

---

## Чек-лист (10 пунктов)

- [ ] **1.1** Git репозиторий инициализирован (.git существует)
- [ ] **1.2** Ветка develop существует
- [ ] **1.3** .gitignore существует и настроен
- [ ] **1.4** README.md существует
- [ ] **1.5** .version существует с корректной версией (SemVer)
- [ ] **1.6** package.json существует (для Node.js проектов)
- [ ] **1.7** .qwen/ директория существует
- [ ] **1.8** .qwen/agents/ директория существует
- [ ] **1.9** .qwen/mcp.json существует
- [ ] **1.10** constitution.md существует (опционально)

---

## Использование

```bash
# Автоматическая проверка
.qwen/checklists/validate-checklist.sh --name "Pre-Flight"

# Интерактивная проверка
.qwen/checklists/checklist-runner.sh "Pre-Flight"

# Проверка с авто-исправлением
.qwen/checklists/validate-checklist.sh --name "Pre-Flight" --auto-fix
```

- ✅ 90-100%: Готово к работе
- ⚠️ 70-89%: Требуется настройка
- ❌ <70%: Критические проблемы
