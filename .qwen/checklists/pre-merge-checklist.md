# Pre-Merge Checklist

Чек-лист проверки перед слиянием ветки.

---

## Чек-лист (8 пунктов)

- [ ] **1.1** Все тесты проходят успешно
- [ ] **1.2** Code review выполнен и одобрен
- [ ] **1.3** CI/CD пайплайн прошел успешно
- [ ] **1.4** Ветка обновлена из base ветки (develop/main)
- [ ] **1.5** Конфликты слияния разрешены
- [ ] **1.6** CHANGELOG обновлен (если требуется)
- [ ] **1.7** Документация обновлена (если требуется)
- [ ] **1.8** Метрики качества не деградировали

---

## Требования к PR

### Описание
- [ ] Четкое описание изменений
- [ ] Ссылка на задачу/issue
- [ ] Скриншоты для UI изменений
- [ ] Примеры использования для API изменений

### Тесты
- [ ] Unit тесты добавлены/обновлены
- [ ] Integration тесты пройдены
- [ ] E2E тесты пройдены (если применимо)

### Безопасность
- [ ] Нет закоммиченных секретов
- [ ] Проверка на уязвимости пройдена
- [ ] Входные данные валидируются

---

## Использование

```bash
# Автоматическая проверка
.qwen/checklists/validate-checklist.sh --name "Pre-Merge"
```
