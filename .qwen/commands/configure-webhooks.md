## Ввод пользователя

```text

```

Вы **ДОЛЖНЫ** учитывать ввод пользователя перед продолжением (если не пустой).

## Описание команды

Команда `/configure-webhooks` помогает настроить систему вебхуков для уведомлений о событиях в проекте. Она предоставляет информацию о формате конфигурации и помогает настроить уведомления.

## Параметры

- `$ARGUMENTS` - необязательные параметры для настройки (например, тип события для настройки)

## Процесс выполнения

1. Проверяет наличие файла конфигурации вебхуков
2. Предоставляет информацию о формате конфигурации
3. Объясняет, как настроить вебхуки для разных типов событий
4. Предоставляет примеры настройки для популярных сервисов

## Возвращаемое значение

Команда возвращает:
- Информацию о формате файла конфигурации
- Примеры настройки вебхуков
- Рекомендации по настройке уведомлений
- Проверку корректности текущей конфигурации
