## Ввод пользователя

```text
$ARGUMENTS
```

Вы **ДОЛЖНЫ** учитывать ввод пользователя перед продолжением (если не пустой).

## Описание команды

Команда `/health-cleanup` запускает процесс обнаружения и удаления мертвого кода в проекте. Она использует агента `dead-code-detector` для сканирования кодовой базы на наличие неиспользуемых компонентов.

## Параметры

- `$ARGUMENTS` - необязательные параметры для настройки анализа (например, конкретные файлы или директории для проверки)

## Процесс выполнения

1. Собирает полный контекст проекта (читает код, ищет паттерны, проверяет связи между файлами)
2. Запускает агента `dead-code-detector` с полным контекстом и критериями валидации
3. Агент анализирует код на наличие мертвого кода
4. Генерирует отчет о найденном мертвом коде
5. Предлагает рекомендации по очистке кода

## Возвращаемое значение

Команда возвращает:
- Отчет о мертвом коде с описанием найденных компонентов
- Рекомендации по очистке мертвого кода
- Приоритезированный список элементов для удаления
- Статистику по типам и количеству найденного мертвого кода
