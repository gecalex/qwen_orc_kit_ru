## Ввод пользователя

```text
$ARGUMENTS
```

Вы **ДОЛЖНЫ** учитывать ввод пользователя перед продолжением (если не пустой).

## Описание команды

Команда `/health-security` запускает полный рабочий процесс обнаружения и исправления уязвимостей безопасности в проекте. Она использует агента `security-analyzer` для сканирования кодовой базы на наличие потенциальных уязвимостей.

## Параметры

- `$ARGUMENTS` - необязательные параметры для настройки анализа (например, конкретные файлы или директории для проверки)

## Процесс выполнения

1. Собирает полный контекст проекта (читает код, ищет паттерны, проверяет коммиты)
2. Запускает агента `security-analyzer` с полным контекстом и критериями валидации
3. Агент анализирует код на наличие уязвимостей
4. Генерирует отчет о найденных уязвимостях
5. Предлагает рекомендации по устранению уязвимостей

## Возвращаемое значение

Команда возвращает:
- Отчет о безопасности с описанием найденных уязвимостей
- Рекомендации по устранению уязвимостей
- Приоритезированный список проблем для исправления
- Статистику по типам и количеству уязвимостей
