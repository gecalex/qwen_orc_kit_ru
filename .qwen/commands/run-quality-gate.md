## Ввод пользователя

```text
$ARGUMENTS
```

Вы **ДОЛЖНЫ** учитывать ввод пользователя перед продолжением (если не пустой).

## Описание команды

Команда `/run-quality-gate` запускает комплексную проверку качества проекта. Она выполняет несколько проверок, включая проверки покрытия тестами, безопасности и размера бандла, и возвращает результаты этих проверок.

## Параметры

- `$ARGUMENTS` - необязательные параметры для настройки проверок (например, конкретные типы проверок для выполнения)

## Процесс выполнения

1. Собирает полный контекст проекта (читает конфигурации, проверяет тип проекта)
2. Запускает агента `code-quality-checker` с полным контекстом и критериями валидации
3. Агент выполняет проверки качества проекта
4. Генерирует отчет о качестве проекта
5. Возвращает статус прохождения контрольных точек качества

## Возвращаемое значение

Команда возвращает:
- Отчет о качестве проекта с результатами всех проверок
- Статус прохождения каждой контрольной точки качества
- Рекомендации по улучшению качества
- Сводку по всем проверкам
