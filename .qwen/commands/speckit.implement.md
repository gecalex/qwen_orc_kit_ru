# SpecKit Command: Implement

**Назначение:** Реализация проекта по плану

**Версия:** 1.0.0

---

## Описание

Команда `speckit.implement` выполняет задачи из плана реализации, делегирует работу специализированным агентам и мониторит прогресс выполнения.

## Использование

```bash
/qwen speckit.implement
```

## Функционал

### 1. Выполнение задач из plan.md
- Последовательное выполнение задач по фазам
- Проверка зависимостей перед началом
- Валидация результатов каждой задачи
- Обновление статуса выполнения

### 2. Делегирование агентам
- Выбор подходящего агента для каждой задачи
- Передача контекста и требований
- Мониторинг выполнения агентом
- Сбор результатов от агентов

### 3. Мониторинг прогресса
- Отслеживание процента выполнения
- Логирование каждого шага
- Обновление прогресс-бара
- Уведомление о завершении фаз

### 4. Управление качеством
- Запуск проверок качества после каждой фазы
- Валидация результатов
- Обработка ошибок
- Откат при неудаче

## Выходные артефакты

- `specs/{ID}/implementation-log.md` - Лог выполнения
- `specs/{ID}/phase-reports/` - Отчеты по фазам
- `specs/{ID}/implementation-summary.md` - Итоговое резюме
- `specs/{ID}/code-changes.md` - Список изменений кода

## Интеграция

**Предыдущая команда:** `speckit.plan`

**Следующая команда:** `speckit.checklist`

**Зависимости:** `specs/{ID}/plan.md`

## Агенты для делегирования

| Тип задачи | Агент | Описание |
|------------|-------|----------|
| Анализ кода | `work_code_analyzer` | Анализ существующего кода |
| Разработка | `work_dev_specialist` | Написание нового кода |
| Тестирование | `work_test_specialist` | Создание тестов |
| Документирование | `work_doc_writer` | Написание документации |
| Рефакторинг | `work_refactor_specialist` | Улучшение кода |

## Пример прогресса

```markdown
# Implementation Progress

## Фаза 0: Подготовка ✅ Завершено
- [x] Задача 0.1: Настройка окружения (15 мин)
- [x] Задача 0.2: Инициализация репозитория (10 мин)
- [x] Задача 0.3: Конфигурация CI/CD (20 мин)

## Фаза 1: Инфраструктура 🔄 В процессе (60%)
- [x] Задача 1.1: Создание базовой структуры
- [~] Задача 1.2: Настройка зависимостей
- [ ] Задача 1.3: Конфигурация линтеров

## Общий прогресс: 45%
## Осталось времени: ~6 часов
```
