## Ввод пользователя

```text
$ARGUMENTS
```

Вы **ДОЛЖНЫ** учитывать ввод пользователя перед продолжением (если не пустой).

## Описание команды

Команда `/worktree-list` показывает список всех созданных worktree в проекте. Это позволяет отслеживать текущие активные ветки разработки.

## Параметры

- `$ARGUMENTS` - необязательные параметры для фильтрации списка

## Процесс выполнения

1. Проверяет, является ли текущий каталог Git-репозиторием
2. Получает список всех worktree в репозитории
3. Показывает информацию о каждом worktree
4. Отмечает активные и неактивные worktree

## Возвращаемое значение

Команда возвращает:
- Список всех worktree с информацией о них
- Статус каждого worktree (активный, неактивный)
- Путь к каждому worktree
