# Интеграция навыков с агентами

## Обзор

Этот документ описывает, как экспериментальные навыки агентов интегрируются с существующей системой агентов в проекте Qwen Code Orchestrator Kit.

## Взаимодействие между навыками и агентами

### Совместное использование

Навыки и агенты могут работать совместно для решения сложных задач:

1. Агент может вызвать навык для выполнения специфической задачи
2. Навык может использовать результаты работы агента
3. Модель может автоматически выбирать между агентом и навыком на основе запроса

### Приоритетность

- Навыки вызываются автоматически моделью на основе описания и контекста
- Агенты вызываются явно через команду `task`
- В случае конфликта, явные вызовы агентов имеют приоритет

## Примеры интеграции

### Пример 1: Комплексный анализ кода

```
Пользователь: "Проанализируй этот код на уязвимости и качество."
```

Возможный процесс:
1. Модель определяет, что задача включает безопасность и качество
2. Автоматически вызываются навыки `security-analyzer` и `code-quality-checker`
3. Результаты объединяются в комплексный отчет

### Пример 2: Использование агента с навыком

```
task '{
  "subagent_type": "bug-hunter",
  "description": "Поиск багов с дополнительным анализом безопасности",
  "prompt": "Проанализируй код и используй навык security-analyzer для дополнительной проверки на уязвимости."
}'
```

## Рекомендации по использованию

### Когда использовать агенты

- Для задач, требующих явного управления и контроля
- Для сложных многошаговых процессов
- Когда необходима гарантия выполнения конкретной задачи

### Когда использовать навыки

- Для автоматических действий на основе контекста
- Для модульных задач, которые могут быть вызваны по необходимости
- Для расширения возможностей модели без явного указания

## Безопасность

- Оба подхода (агенты и навыки) следуют принципу минимальных привилегий
- Навыки ограничены в возможностях по сравнению с агентами
- Все действия регистрируются для аудита

## Будущее развитие

- Возможность создания цепочек из навыков
- Совместное использование контекста между агентами и навыками
- Улучшенная интеграция с системой контрольных точек качества
