## Технические особенности

```
.qwen/
├── agents/                 # Агенты (оркестраторы и воркеры)
├── skills/                 # Навыки для часто используемых операций
├── templates/              # Шаблоны для создания компонентов
└── configs/                # Конфигурации для MCP и других сервисов

.qwen/scripts/
├── qwen-cli-specialist/    # Скрипты агента-специалиста
├── validation/             # Скрипты проверки соответствия
└── monitoring/             # Скрипты мониторинга

.qwen/docs/
└── help/
    └── qwen_orchestrator_kit/      # Документация по Qwen Orchestrator Kit
```
