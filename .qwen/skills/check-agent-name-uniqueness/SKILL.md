# Check Agent Name Uniqueness

## Когда использовать
- Перед созданием нового агента
- При валидации имени агента
- Для предотвращения конфликтов имен
- В процессе регистрации новых агентов

## Инструкции
1. Получите имя агента для проверки
2. Проверьте существующие файлы агентов в .qwen/agents/
3. Сравните имя с существующими именами
4. Верните результат проверки
