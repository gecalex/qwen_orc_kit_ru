# Code Cleaner

## Когда использовать
- При удалении мертвого кода
- Для удаления неиспользуемых импортов
- При очистке неиспользуемых переменных
- Для удаления закомментированного кода
- При рефакторинге устаревшего кода

## Инструкции
1. Проанализируйте код на наличие мертвого кода
2. Определите неиспользуемые компоненты
3. Удалите мертвый код с осторожностью
4. Проверьте, что удаление не нарушает функциональность
5. Обновите документацию при необходимости
