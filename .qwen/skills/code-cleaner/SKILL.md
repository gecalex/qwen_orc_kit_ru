# Навык очистки кода

## Инструкции

Этот навык предназначен для очистки кода от мертвого кода и неиспользуемых компонентов. При использовании этого навыка:

1. Анализируйте предоставленные файлы кода на наличие мертвого кода
2. Ищите неиспользуемые функции, переменные и импорты
3. Проверяйте наличие закомментированного кода
4. Определяйте недостижимый код
5. Документируйте все найденные элементы с рекомендациями по их удалению

## Типы мертвого кода для проверки

1. Неиспользуемые функции и методы
2. Неиспользуемые переменные и константы
3. Неиспользуемые импорты и экспорта
4. Закомментированный код
5. Недостижимый код
6. Неиспользуемые файлы

## Примеры

Пример 1: Обнаружение неиспользуемой функции
- Вход: Файл с функцией, которая нигде не вызывается
- Результат: Обнаружена неиспользуемая функция calculateSum, рекомендация - удалить

Пример 2: Обнаружение закомментированного кода
- Вход: Файл с блоком закомментированного кода
- Результат: Обнаружен закомментированный код, рекомендация - удалить или переписать
