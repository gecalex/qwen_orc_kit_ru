# Dependency Auditor

## Когда использовать
- При анализе зависимостей проекта
- Для проверки наличия уязвимых зависимостей
- При проверке актуальности версий зависимостей
- Для выявления неиспользуемых зависимостей
- При анализе конфликтов зависимостей

## Инструкции
1. Проанализируйте файлы зависимостей проекта
2. Проверьте зависимости на наличие уязвимостей
3. Определите устаревшие зависимости
4. Найдите неиспользуемые зависимости
5. Сформируйте отчет о состоянии зависимостей
