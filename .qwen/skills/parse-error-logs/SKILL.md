## Когда использовать
- При обработке вывода сборки

- Для анализа логов проверки типов
- При диагностике проблем в процессе CI/CD

1. Получите логи ошибок для анализа
2. Определите тип ошибок (сборка, тесты, типизация и т.д.)
3. Извлеките ключевую информацию из логов
4. Структурируйте данные об ошибках
5. Сформируйте отчет о найденных ошибках
