# Resume Session

## Когда использовать
- При продолжении работы из предыдущих сессий
- Для восстановления прерванных рабочих процессов
- При перезапуске после ошибок
- Для обеспечения непрерывности между сессиями
- При возврате управления от других агентов

## Инструкции
1. Найдите сохраненное состояние сессии
2. Загрузите состояние выполнения
3. Восстановите прогресс и контекст
4. Продолжите выполнение с места остановки
5. Обновите состояние сессии
