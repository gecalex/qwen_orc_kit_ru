# Run Quality Gate

## Когда использовать
- При реализации контрольных точек качества
- Для проверки соответствия стандартам
- В процессах CI/CD перед деплоем
- При проверке качества кода
- Для автоматизации проверок качества

## Инструкции
1. Определите тип проверки качества
2. Выполните соответствующую проверку
3. Оцените результат проверки
4. Примените настройки блокировки при необходимости
5. Сформируйте результат проверки
