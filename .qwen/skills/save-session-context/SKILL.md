# Save Session Context

## Когда использовать
- При завершении фазы выполнения
- Перед окончанием сессии
- При переключении контекстов
- Для обеспечения непрерывности между сессиями
- При возврате управления другим агентам

## Инструкции
1. Соберите текущее состояние выполнения
2. Сохраните прогресс и контекст
3. Запишите информацию в файл состояния
4. Подготовьте данные для восстановления
5. Подтвердите успешное сохранение
