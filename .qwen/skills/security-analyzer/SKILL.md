# Security Analyzer

## Когда использовать
- При проверке кода на уязвимости
- Для анализа безопасности приложений
- При проверке на SQL-инъекции
- Для поиска XSS уязвимостей
- При проверке на CSRF уязвимости
- Для анализа безопасности аутентификации

## Инструкции
1. Проанализируйте код на наличие уязвимостей
2. Проверьте обработку пользовательского ввода
3. Оцените безопасность аутентификации
4. Проверьте защиту от известных атак
5. Сформируйте отчет о безопасности
