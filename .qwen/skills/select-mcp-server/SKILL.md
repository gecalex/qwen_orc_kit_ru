# Select MCP Server

## Когда использовать
- При выборе подходящих серверов MCP для задач
- Для определения правильного сервера на основе типа задачи
- При интеграции с внешними сервисами через MCP
- Для оптимизации использования ресурсов MCP
- При маршрутизации задач к специализированным серверам

## Инструкции
1. Определите тип задачи и требования
2. Найдите подходящие серверы MCP
3. Проверьте доступность серверов
4. Выберите оптимальный сервер
5. Верните рекомендацию по использованию сервера
