# Шаблон навыка агента

## Инструкции

Этот шаблон служит основой для создания новых навыков агента. При создании нового навыка:

1. Скопируйте эту структуру в новую директорию в `.qwen/skills/`
2. Измените `name` и `description` в YAML заголовке
3. Обновите название и инструкции в соответствии с назначением навыка
4. Добавьте примеры использования
5. Укажите требования к безопасности

## Требования к безопасности

- Опишите, какие инструменты использует навык и почему
- Укажите, какие меры предосторожности применяются
- Ограничьте использование потенциально опасных операций

## Примеры

Пример 1: Описание задачи
- Пользователь: "Пример запроса пользователя"
- Результат: "Описание ожидаемого результата"

Пример 2: Другой сценарий использования
- Пользователь: "Еще один пример запроса"
- Результат: "Описание другого ожидаемого результата"
