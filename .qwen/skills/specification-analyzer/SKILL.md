# Specification Analyzer

## Когда использовать
- При проверке соответствия реализации спецификациям
- Для анализа требований в спецификациях
- При проверке полноты спецификаций
- Для выявления несоответствий между спецификацией и реализацией
- При ревью спецификаций

## Инструкции
1. Прочитайте спецификацию
2. Проанализируйте требования
3. Сравните с текущей реализацией
4. Выявите несоответствия
5. Сформируйте отчет о соответствии
