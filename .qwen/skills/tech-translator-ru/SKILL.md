# Навык технического перевода на русский

## Инструкции

Этот навык предназначен для перевода технической документации, комментариев в коде, докстрингов и сообщений с английского на русский язык. При использовании этого навыка:

1. Анализируйте предоставленный контент для перевода
2. Определяйте технический контекст и домен
3. Переводите текст, сохраняя техническую точность
4. Проверяйте, что перевод понятен и соответствует стилю
5. Сохраняйте оригинальное форматирование и структуру

## Требования к безопасности

- Используйте только read_file и write_file для безопасной работы с файлами
- НЕ используйте run_shell_command, web_search, web_fetch или task для получения или изменения внешних данных

## Примеры

Пример 1: Перевод комментариев в коде
- Пользователь: "Переведи комментарии в файле src/utils/helpers.py с английского на русский, сохранив форматирование и техническую точность."
- Результат: Файл src/utils/helpers.py с переведенными на русский язык комментариями, сохраняющими техническую точность и оригинальное форматирование

Пример 2: Перевод документации
- Пользователь: "Переведи README.md файл на русский язык, сохранив структуру и форматирование markdown."
- Результат: Файл README.md с переведенным на русский язык содержимым, сохраняющим структуру и форматирование markdown
