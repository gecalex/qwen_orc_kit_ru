# Навык отправки вебхуков

## Инструкции

Этот навык предназначен для отправки вебхуков (HTTP-запросов) в ответ на события в системе. При использовании этого навыка:

1. Определите тип события, которое вызывает вебхук
2. Укажите URL для отправки уведомления
3. Сформируйте тело запроса в соответствии с шаблоном
4. Отправьте HTTP-запрос с использованием curl или другого инструмента
5. Обработайте результат отправки

## Параметры вебхука

1. event_type: тип события, которое вызывает вебхук
2. url: URL для отправки уведомления
3. headers: HTTP-заголовки для запроса
4. payload_template: шаблон тела запроса

## Примеры

Пример 1: Отправка уведомления о завершении задачи
- Вход: Событие "task_completed", URL "https://hooks.slack.com/...", шаблон пейлоада
- Результат: HTTP-запрос отправлен, статус отправки возвращен

Пример 2: Отправка уведомления об ошибке
- Вход: Событие "error_occurred", URL "https://discord.com/api/webhooks/...", шаблон пейлоада
- Результат: HTTP-запрос отправлен, статус отправки возвращен
