## Ожидающие создания

| Агент | Домен | Тип | Задачи | Приоритет | Статус | Создан |
|-------|-------|-----|--------|-----------|--------|--------|
| - | - | - | - | - | - | - |

**Легенда статусов:**
- ⏳ Ожидает — агент ещё не создан
- 🔄 В процессе — agent creation запущен
- ✅ Создан — агент создан и готов
- ❌ Ошибка — не удалось создать

**Легенда приоритетов:**
- P0: Критический (блокирует задачи)
- P1: Важный (нужен для MVP)
- P2: Желательный (nice to have)

## Инструкция по добавлению

Когда `work_planning_agent_requirer` выявляет отсутствующего агента:

1. Добавить запись в таблицу "Ожидающие создания"
2. Назначить приоритет (P0/P1/P2)
3. Запустить `work_dev_meta_agent` для создания
4. Обновить статус после создания
5. Добавить запись в "Журнал создания"
