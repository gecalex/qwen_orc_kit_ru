# Установка Qwen Code Orchestrator Kit

Данный документ описывает процесс установки и начальной настройки Qwen Code Orchestrator Kit.

## Требования

- Git
- Python 3.11 или выше
- Node.js (по желанию, для JavaScript/TypeScript проектов)
- Установленный Qwen Code CLI

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/ваш-репозиторий/qwen_orc_kit_ru.git
cd qwen_orc_kit_ru
```

### 2. Установка зависимостей

Для Python проектов:

```bash
# Установка uv (если не установлен)
pip install uv

# Установка зависимостей проекта
uv sync --dev
```

Для JavaScript/TypeScript проектов:

```bash
# Установка зависимостей
npm install
# или
pnpm install
```

### 3. Настройка Qwen CLI

1. Убедитесь, что Qwen Code CLI установлен в вашей системе
2. Установите MCP-сервера, если они требуются для вашего проекта
3. Настройте `.qwen/settings.json` с необходимыми параметрами

### 4. Проверка установки

Для проверки корректности установки выполните:

```bash
# Проверка доступных агентов
ls .qwen/agents/
```

Вы должны увидеть следующие файлы агентов:
- `bug-fixer.md`
- `bug-hunter.md`
- `code-quality-checker.md`
- `security-orchestrator.md`
- `tech-translator-ru.md`

## Настройка среды разработки

### VS Code

Для работы с проектом рекомендуется использовать VS Code с установленными расширениями:

- Python extension pack
- ESLint (для JS/TS проектов)
- Prettier (для форматирования)

### PyCharm/IntelliJ IDEA

Для Python проектов можно использовать PyCharm с плагинами:
- Python
- IdeaVim (если используете vim режим)

## Запуск первого задания

После установки вы можете запустить первое задание через оркестратор:

```bash
# Пример запуска агента
task '{
  "subagent_type": "bug-hunter",
  "description": "Поиск потенциальных проблем в коде",
  "prompt": "Проанализируй файлы в директории src/ на предмет потенциальных багов и уязвимостей."
}'
```

## Устранение неполадок

### Проблемы с запуском агентов

Если агенты не запускаются, проверьте:

1. Корректность YAML frontmatter в файлах агентов
2. Наличие всех необходимых инструментов в системе
3. Правильность синтаксиса в файле задачи

### Проблемы с доступом к файлам

Убедитесь, что:

1. У вас есть права на чтение/запись в директории проекта
2. Все пути к файлам указаны корректно
3. Файлы агентов находятся в директории `.qwen/agents/`
