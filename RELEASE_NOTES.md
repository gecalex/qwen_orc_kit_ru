# Релиз v0.2.0 - Подготовка проекта Qwen Orchestrator Kit к релизу

## Описание
Этот релиз включает в себя подготовленную версию проекта Qwen Orchestrator Kit, готовую к использованию. В релиз включены только те компоненты, которые необходимы для выполнения основных задач проекта, без файлов и каталогов разработки.

## Что нового
- Подготовлена релизная версия проекта с исключением файлов разработки
- Включены все основные компоненты: агенты, навыки, шаблоны, документация и скрипты
- Обновлены файлы .gitignore для правильного исключения файлов разработки из будущих релизов
- Создана система индексации и навигации по агентам
- Реализована система навыков для переиспользования функций
- Включены MCP-конфигурации для интеграции с внешними инструментами
- Реализована автоматическая маршрутизация задач подходящим агентам
- Добавлена система уведомлений через вебхуки
- Создана система стандартизированных отчетов
- Включена автоматизированная система тестирования

## Компоненты проекта
- `.qwen/agents/` - Агенты системы (оркестраторы и воркеры)
- `.qwen/skills/` - Навыки для переиспользования функций
- `.qwen/templates/` - Шаблоны для создания новых компонентов
- `docs/architecture/` - Архитектурная документация
- `scripts/` - Скрипты для работы системы
- `.mcp.*.json` - Конфигурации MCP для интеграции с внешними инструментами
- `QWEN.md` - Основной файл контекста агента
- `README.md` - Основная документация
- `LICENSE` - Лицензия проекта

## Исключенные компоненты
- `/specs/` - Директории спецификаций (для разработки)
- `/research/` - Исследовательские файлы
- `/plans/` - Планы реализации
- `/tasks/` - Файлы задач
- Временные файлы и черновики
- Файлы промежуточной разработки

## Совместимость
Этот релиз совместим с Qwen Code CLI и поддерживает плоскую структуру агентов, как предусмотрено архитектурой проекта.

## Известные проблемы
- Нет известных проблем, влияющих на основную функциональность

- Создание новых агентов с помощью мета-агента
