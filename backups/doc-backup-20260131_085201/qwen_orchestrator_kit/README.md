# Документация по Qwen Orchestrator Kit

Добро пожаловать в директорию документации по Qwen Orchestrator Kit. Здесь вы найдете всю необходимую информацию для эффективной разработки с использованием системы агентов Qwen Orchestrator Kit.

## Структура документации

- [Введение в Qwen Orchestrator Kit](./getting-started.md) — основы работы с системой
- [Стандарты YAML заголовков](./yaml-standards.md) — требования к оформлению компонентов
- [Рекомендации по шаблонам](./template-guidelines.md) — создание и использование шаблонов
- [Процессы обеспечения качества](./quality-assurance.md) — контрольные точки и проверки
- [Интеграция с MCP](./mcp-integration.md) — использование внешних сервисов
- [Разработка агентов](./agent-development/index.md) — руководство по созданию агентов
- [Разработка навыков](./skill-development/index.md) — руководство по созданию навыков
- [Инструменты автоматизации](./automation-tools/index.md) — автоматизация проверок и процессов

## Назначение

Эта документация предназначена для:
- Разработчиков, создающих агентов и навыки
- Архитекторов, проектирующих системы на основе Qwen Orchestrator Kit
- QA-инженеров, проверяющих качество компонентов
- Технических писателей, документирующих проекты

## Актуальность

Документация регулярно проверяется и обновляется с помощью агента-специалиста `qwen-code-cli-specialist`, который отслеживает соответствие официальному руководству Qwen Orchestrator Kit и автоматизирует проверки соответствия.

## Вклад в развитие

Если вы нашли ошибку или хотите дополнить документацию:
1. Создайте issue с описанием проблемы
2. Предложите улучшение через pull request
3. Обсудите изменения с командой разработчиков
