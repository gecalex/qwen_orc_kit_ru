Выполнить скрипт автоматизации релиза с автоподтверждением для Claude Code.

**Возможности:**
- Автоматическая синхронизация версий в package.json с последним git-тегом (предотвращает конфликты версий)
- Анализ коммитов с момента последнего релиза
- Автоматическое определение типа обновления версии из conventional commits
- Генерация записей в CHANGELOG
- Обновление всех файлов package.json
- Создание git-тега и отправка в GitHub
- Полная поддержка отката при ошибках

**Использование:**

# Сначала перейдите в корневую директорию проекта
PROJECT_ROOT=$(git rev-parse --show-toplevel 2>/dev/null || echo "$PWD")
cd "$PROJECT_ROOT" && bash .claude/scripts/release.sh $ARGUMENTS --yes
