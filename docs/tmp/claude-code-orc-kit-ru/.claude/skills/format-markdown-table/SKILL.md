# Форматирование таблицы markdown

Создание правильно отформатированных таблиц markdown с выравниванием и последовательными интервалами.

## Когда использовать

- Таблицы статистики отчетов
- Сравнительные таблицы
- Представление сводных данных
- Панели метрик в отчетах

## Инструкции

### Шаг 1: Получение данных таблицы

Принять заголовки и строки в качестве входных данных.

**Ожидаемый ввод**:
```json
{
  "headers": ["Столбец 1", "Столбец 2", "Столбец 3"],
  "rows": [
    ["Значение 1", "Значение 2", "Значение 3"],
    ["Значение 4", "Значение 5", "Значение 6"]
  ],
  "alignment": ["left", "center", "right"]
}
```

**Выравнивание** (необязательно):
- `left`: Выравнивание по левому краю (по умолчанию)
- `center`: Выравнивание по центру
- `right`: Выравнивание по правому краю

### Шаг 2: Расчет ширины столбцов

Определить максимальную ширину для каждого столбца.

**Расчет ширины**:
- Включить ширину заголовка
- Включить все значения строк
- Добавить отступ (1 пробел с каждой стороны)
- Минимальная ширина: 3 символа

### Шаг 3: Форматирование строки заголовка

Создать строку заголовка с правильными интервалами.

**Формат**:
```
| Заголовок 1 | Заголовок 2 | Заголовок 3 |
```

### Шаг 4: Форматирование строки-разделителя

Создать разделитель с индикаторами выравнивания.

**Индикаторы выравнивания**:
- Слева: `:---` или `---`
- По центру: `:---:`
- Справа: `---:`

**Формат**:
```
|----------|:--------:|---------:|
```

### Шаг 5: Форматирование строк данных

Создать строки данных с последовательными интервалами.

**Формат**:
```
| Значение 1 | Значение 2 | Значение 3 |
| Значение 4 | Значение 5 | Значение 6 |
```

### Шаг 6: Возврат полной таблицы

Вернуть отформатированную таблицу markdown.

**Ожидаемый вывод**:
```markdown
| Столбец 1 | Столбец 2 | Столбец 3 |
|----------|:--------:|---------:|
| Значение 1 | Значение 2 | Значение 3 |
| Значение 4 | Значение 5 | Значение 6 |
```

## Обработка ошибок

- **Пустые заголовки**: Вернуть ошибку с запросом заголовков
- **Пустые строки**: Вернуть предупреждение, создать таблицу только с заголовками
- **Несоответствие столбцов**: Заполнить короткие строки пустыми ячейками
- **Неверное выравнивание**: Использовать 'left' по умолчанию, выдать предупреждение

## Примеры

### Пример 1: Простая таблица статистики

**Ввод**:
```json
{
  "headers": ["Метрика", "Количество", "Процент"],
  "rows": [
    ["Критические ошибки", "3", "13%"],
    ["Высокие ошибки", "8", "35%"],
    ["Средние ошибки", "12", "52%"]
  ],
  "alignment": ["left", "right", "right"]
}
```

**Вывод**:
```markdown
| Метрика         | Количество | Процент    |
|-----------------|-----------:|-----------:|
| Критические ошибки |          3 |        13% |
| Высокие ошибки     |          8 |        35% |
| Средние ошибки     |         12 |        52% |
```

### Пример 2: Сравнительная таблица

**Ввод**:
```json
{
  "headers": ["Функция", "До", "После"],
  "rows": [
    ["Время сборки", "45с", "12с"],
    ["Время теста", "2м 30с", "1м 15с"],
    ["Размер бандла", "2.3 МБ", "1.8 МБ"]
  ],
  "alignment": ["left", "center", "center"]
}
```

**Вывод**:
```markdown
| Функция       |   До    |  После  |
|---------------|:-------:|:-------:|
| Время сборки  |   45с   |   12с   |
| Время теста   | 2м 30с  | 1м 15с  |
| Размер бандла | 2.3 МБ  | 1.8 МБ  |
```

### Пример 3: Распределение приоритетов

**Ввод**:
```json
{
  "headers": ["Приоритет", "Открыто", "Исправлено", "Всего"],
  "rows": [
    ["P0", "2", "5", "7"],
    ["P1", "5", "3", "8"],
    ["P2", "12", "8", "20"]
  ]
}
```

**Вывод**:
```markdown
| Приоритет | Открыто | Исправлено | Всего |
|-----------|---------|------------|-------|
| P0        | 2       | 5          | 7     |
| P1        | 5       | 3          | 8     |
| P2        | 12      | 8          | 20    |
```

### Пример 4: Пустые строки (только заголовок)

**Ввод**:
```json
{
  "headers": ["Имя", "Значение", "Статус"],
  "rows": []
}
```

**Вывод**:
```markdown
| Имя | Значение | Статус |
|-----|----------|--------|
```

## Валидация

- [ ] Правильно форматирует заголовки
- [ ] Создает правильные разделители
- [ ] Выравнивает столбцы как указано
- [ ] Обрабатывает различные типы данных (числа, текст)
- [ ] Заполняет столбцы для последовательной ширины
- [ ] Корректно обрабатывает пустые строки
