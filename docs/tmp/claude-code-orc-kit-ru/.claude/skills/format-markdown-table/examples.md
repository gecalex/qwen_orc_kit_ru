# Примеры форматирования таблиц Markdown

## Базовая таблица

```markdown
| Заголовок 1 | Заголовок 2 | Заголовок 3 |
|-------------|-------------|-------------|
| Ячейка 1    | Ячейка 2    | Ячейка 3    |
| Ячейка 4    | Ячейка 5    | Ячейка 6    |
```

## Примеры выравнивания

### Выравнивание по левому краю (по умолчанию)
```markdown
| Имя    | Возраст | Город     |
|--------|---------|-----------|
| Алиса  | 30      | Нью-Йорк  |
| Боб    | 25      | Бостон    |
```

### Выравнивание по центру
```markdown
| Имя    | Возраст | Город     |
|:------:|:-------:|:---------:|
| Алиса  | 30      | Нью-Йорк  |
| Боб    | 25      | Бостон    |
```

### Выравнивание по правому краю
```markdown
| Имя    | Возраст | Город     |
|-------:|--------:|----------:|
| Алиса  | 30      | Нью-Йорк  |
| Боб    | 25      | Бостон    |
```

### Смешанное выравнивание
```markdown
| Имя    | Возраст | Город     |
|--------|--------:|:---------:|
| Алиса  | 30      | Нью-Йорк  |
| Боб    | 25      | Бостон    |
```

## Сложные таблицы

### С числами
```markdown
| Метрика          | Значение | Изменение |
|------------------|---------:|----------:|
| Доход            | $2.5M    |     +15%  |
| Пользователи     | 50K      |     +23%  |
| Удержание        | 85%      |      +3%  |
```

### С кодом
```markdown
| Функция          | Возвращает | Описание              |
|------------------|------------|----------------------|
| `parse()`        | Object     | Парсит JSON строку   |
| `stringify()`    | String     | Преобразует в JSON   |
| `validate()`     | Boolean    | Проверяет валидность |
```

### С индикаторами статуса
```markdown
| Задача            | Статус | Приоритет |
|-------------------|:------:|:---------:|
| Исправить баг #123|   ✅   |    P0     |
| Обновить документацию | 🔄   |    P2     |
| Рефакторинг API   |   ❌   |    P1     |
```

## Советы

1. **Постоянная ширина**: Используйте пробелы для визуального выравнивания колонок в исходном коде
2. **Минимальная ширина**: Каждая колонка должна быть шириной не менее 3 символов
3. **Отступы**: Добавляйте по 1 пробелу с каждой стороны содержимого ячейки
4. **Заголовки**: Делайте заголовки краткими и понятными
5. **Выравнивание**: Выбирайте выравнивание в зависимости от типа содержимого:
   - Текст: выравнивание по левому краю
   - Числа: выравнивание по правому краю
   - Статусы/Иконки: выравнивание по центру
