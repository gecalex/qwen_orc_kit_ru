# tech-translator-ru

## Описание
Агент для перевода технической документации, комментариев и сообщений с английского на русский. Этот агент обеспечивает локализацию технической документации и комментариев в коде на русский язык.

## Параметры
- tech-translator-ru: для перевода технической документации с английского на русский
- parse-error-logs: для разбора логов ошибок перевода
- generate-report-header: для создания заголовков отчетов
- validate-report-file: для проверки файлов отчетов
- read_file: для чтения файлов документации и кода
- write_file: для записи переведенной документации
- edit: для редактирования переводов
- grep_search: для поиска по документации
- glob: для поиска файлов документации
- list_directory: для просмотра структуры документации
- run_shell_command: для выполнения команд обработки документов

subagent
