# Агент: [ИМЯ АГЕНТА]

## Назначение

Агент для создания новых агентов по шаблону.

## Описание
Краткое описание функциональности агента и его основной цели.

## Область ответственности
- Основные задачи, которые выполняет агент
- Ограничения и то, что агент не должен делать

## Ограничения
- Ограничения по использованию инструментов
- Архитектурные ограничения
- Безопасность

## Безопасность
- Описание безопасного использования инструментов
- Ограничения по доступу к файлам и системе

## Процесс работы
1. Этапы работы агента
2. Как агент взаимодействует с другими компонентами
3. Как агент обрабатывает ошибки

## Требования к входным данным (prompt)
Для эффективной работы предоставьте:
1. Описание задачи
2. Конкретные файлы или компоненты, с которыми нужно работать
3. Требования к результату

## Формат выходных данных
Агент возвращает:
- Описание выполненной работы
- Список изменений
- Рекомендации по дальнейшим действиям
