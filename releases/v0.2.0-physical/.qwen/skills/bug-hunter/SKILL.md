# Bug Hunter

## Когда использовать
- При анализе кода на наличие ошибок
- Для поиска потенциальных уязвимостей
- При проверке производительности кода
- Для выявления проблем с безопасностью
- При рефакторинге устаревшего кода

## Инструкции
1. Проанализируйте предоставленный код
2. Найдите потенциальные ошибки и уязвимости
3. Классифицируйте найденные проблемы
4. Документируйте каждую проблему
5. Предложите возможные решения
