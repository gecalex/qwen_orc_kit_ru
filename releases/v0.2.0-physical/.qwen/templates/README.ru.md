# Доступные шаблоны

В этой директории содержатся шаблоны для различных компонентов системы на русском языке.

## Шаблоны агентов

- `orchestrator-template.md` - Шаблон для оркестраторов
- `worker-template.md` - Шаблон для воркеров
- `report-template.md` - Шаблон отчетов

## Шаблоны навыков

- `skill-template.md` - Шаблон для создания новых навыков
- В директории `.qwen/skills/template/SKILL.md` - Альтернативный шаблон навыков

## Использование

При создании новых компонентов вы можете использовать соответствующие шаблоны для создания русскоязычных агентов и навыков.
