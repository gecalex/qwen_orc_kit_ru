## 📋 ОБЗОР

Создана полноценная система переключения между двумя режимами работы:
- **Быстрый старт (quickstart)** — минимальный набор для начала работы
- **Полный шаблон (full)** — все возможности системы

---

## 🎯 РЕАЛИЗОВАННЫЕ КОМПОНЕНТЫ

### **1. Скрипт переключения**

**Файл:** `.qwen/scripts/template-switcher.sh`

**Возможности:**
- ✅ Переключение в режим быстрого старта
- ✅ Переключение в полный режим
- ✅ Проверка текущего статуса
- ✅ Автоматическое резервное копирование
- ✅ Кэширование полного шаблона
- ✅ Подробная статистика
- ✅ Цветной вывод
- ✅ Обработка ошибок

**Команды:**
```bash
bash .qwen/scripts/template-switcher.sh quickstart  # Быстрый старт
bash .qwen/scripts/template-switcher.sh full        # Полный шаблон
bash .qwen/scripts/template-switcher.sh status      # Статус
bash .qwen/scripts/template-switcher.sh help        # Справка
```

---

**Использование:**
```bash

```

**Алиасы:**
- `qs`, `q` — быстрый старт
- `f` — полный шаблон
- `s`, `stat` — статус

## 🔧 КАК ЭТО РАБОТАЕТ

### **Быстрый старт (алгоритм):**

1. Проверка текущего режима
2. Создание резервной копии (`.qwen/.backup-YYYYMMDD-HHMMSS/`)
3. Удаление специализированных агентов (остается 7)
4. Удаление специализированных навыков (остается 10)
5. Удаление специализированных команд (остается 5)
6. Удаление специализированных скриптов (остается 3)
7. Показ статистики

**Результат:**
- 7 агентов (базовые оркестраторы + bug-fixer/hunter)
- 10 навыков (валидация, безопасность, анализ)
- 5 команд (Quality Gates, security, orchestration)
- 3 скрипта (анализ состояния, security, bugs)

---

### **Полный шаблон (алгоритм):**

1. Проверка текущего режима
2. Проверка кэша (`.qwen/template-cache/`)
3. Если кэша нет → создание из текущих файлов
4. Восстановление всех файлов из кэша
5. Показ статистики

**Результат:**
- 25 агентов (все оркестраторы и воркеры)
- 31 навык (все категории)
- 22 команды (speckit, health, worktree, другие)
- ~20 скриптов (все категории)

## 🆘 УСТРАНЕНИЕ ПРОБЛЕМ

### **Проблема 1: Скрипт не находит кэш**

**Решение:**
```bash
# Создайте кэш вручную
mkdir -p .qwen/template-cache
cp -r .qwen/agents/* .qwen/template-cache/agents/
cp -r .qwen/skills/* .qwen/template-cache/skills/
# ... и так далее
```

---

### **Проблема 2: Ошибка при переключении**

**Решение:**
1. Проверьте права на файлы
2. Убедитесь, что ни один файл не открыт в редакторе
3. Восстановите из резервной копии

---

### **Проблема 3: Пропали нужные агенты**

**Решение:**
```bash
# Переключитесь на полный режим
qwen template-switcher full

# Или восстановите из резервной копии
cp -r .qwen/.backup-*/agents/* .qwen/agents/
```
